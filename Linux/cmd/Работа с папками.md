Создать папку:
```
mkdir [NAME]
```
Создать файл:
```
touch [NAME]
```
Удаление файла:
```
rm [NAME]
```
Удаление каталога:
```
rm -r [NANE]
```
Переименование:
```
mv имя_файла новое_имя
```
Перемещение:
```
mv имя_файла путь/
```
Копирование файла:
```
cp file1 file2
```
